### used tech skills 
React JS , Node JS, Express JS, MongoDB, Socket.io 
## 1. To clone the repo 
git clone git@github.com:vishswati1011/chatApp.git

## 2. to install dependency
  cd server
  npm i 
  cd..
  cd client
  npm i
## 3. To run backend 
  cd server 
  npm start
## 4. To run frontend
  cd client
  npm start

###  How it work
  first you need to signup then 
  invite a user for chat
